# 🎯 Planificador de Cultos - IBEG Chiquimula

**Aplicación web profesional para planificación de servicios religiosos** - Versión HTML Pura (Sin React)

## ✨ Características

- 📅 **Planificación completa de cultos**
- 🎵 **Gestión avanzada de adoración** (hasta 7-8 canciones)
- 👥 **Asignación de voluntarios**
- 📄 **Generación automática de PDF**
- 📱 **100% responsive (móvil)**
- 💾 **Datos guardados localmente**

## 🚀 Despliegue en Vercel (Super Simple)

### Requisitos:
- Cuenta GitHub (gratis)
- Cuenta Vercel (gratis)

### Pasos:

#### 1️⃣ **Descarga todos los archivos**
- README.md
- package.json
- vercel.json
- index.html

#### 2️⃣ **Sube a GitHub**
- Crea repositorio en https://github.com/new
- Nombre: `planificador-cultos-ibeg`
- Sube los 4 archivos

#### 3️⃣ **Conecta a Vercel**
- Ve a https://vercel.com/new
- Importa tu repositorio de GitHub
- Vercel despliega automáticamente ✅

#### 4️⃣ **¡Listo!** 
Tu URL estará lista en 30 segundos

## 💾 Datos

- Los datos se guardan **en tu navegador** (localStorage)
- **Respaldo:** Descarga PDFs regularmente
- **Privado:** Nadie puede ver tus datos

---

## 📝 Cómo Usar

1. Haz clic en **"+ Nuevo Culto"**
2. Completa todos los detalles
3. Agrega canciones con roles específicos
4. Asigna voluntarios
5. Haz clic en **"✓ Guardar Culto"**
6. Descarga PDF haciendo clic en **"📥 PDF"**

---

## 🎨 Características Detalladas

### Secciones del Culto
- **Bienvenida** (10 min)
- **Adoración** (35-40 min, múltiples canciones)
- **Ofrenda** (3 servidores)
- **Predicación** (55-60 min)
- **Consagración** (10 min)

### Roles por Canción
- Líder de Adoración
- Voz Líder
- Responsable
- Voces Secundarias

### Personal de Apoyo
- Sonido
- Proyección de Medios

### Eventos Especiales
- Santa Cena (mensual)
- Bautizos (cada 3-4 meses)
